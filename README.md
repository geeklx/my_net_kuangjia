### 支持视频播放小窗和大窗
### 支持几乎所有的bannerview
### 支持自定义网络布局切换
### 支持自定义虚化背景
### 支持自定义loading
### 支持自定义Tablayout
### 支持自定义隐藏键盘
### 支持自定义MVP
### 支持多版本打版
### 支持网络访问和业务分离
